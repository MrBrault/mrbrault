Task 2: Photos

This is a website I created as a way for students to be able to learn at their own pace which frees me up to work 1-on-1 with the students who need additional help. These are used as an introduction to the content before tackling more challenging tasks.

The original photo could be used as an example of Rule of Space for the unit on capturing quality photographs. I edited all of the photos by using the Mac app, Preview. I used the marquee tool and chose the portion of the image I wanted to keep, and then pressed command + K to crop the image and saved a copy of the new photo. I also resized the original by going to the Tools menu and choosing Adjust Size and set the width to 640 pixels. I made the original black and white by going to Tools and Adjust Color, then dragged the saturation level all the way to the left which leaves no color.

The outside photo was taken by one of the other photography teachers, Susana Butterworth, of our colleague, Brent Rust. This is an example of a Triptych photo edit and would be used to teach that editing technique.

The scanned image is a picture of a camera sign-out sheet that all photography students need to sign. This could be used to show students what they need to complete when checking out a camera. I used the scan feature on my iPhone that is built into the notes app and saved it as an image file. I also resized the scanned image by going to the Tools menu and choosing Adjust Size and set the width to 640 pixels.

The paint image was done on my iPad with the Notes app. I drew an image that could be used as a banner image on the website using different pen sizes and colors. I used the Apple Pencil to draw the image and then saved it when finished. I also resized the paint image by going to the Tools menu and choosing Adjust Size and set the width to 640 pixels.
